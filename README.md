# Muhammad Taha — Premium Developer Portfolio

A premium, fully responsive personal portfolio website built with modern web technologies. Features a cyberpunk/tech-inspired design with glassmorphism, particle backgrounds, animated cursor, and rich micro-animations.

## 🚀 Live Preview

Open `index.html` in your browser to view the portfolio.

## ✨ Features

- **Cyberpunk Design** — Dark theme with neon cyan, purple, and pink accents
- **Glassmorphism** — Frosted glass effects throughout the UI
- **Particle Background** — Canvas-based floating particles with connection lines
- **Custom Animated Cursor** — Dot + outline cursor with hover states
- **Dark / Light Mode** — Toggle with localStorage persistence
- **Typing Animation** — Rotating professional titles
- **Scroll Animations** — IntersectionObserver-powered reveal effects
- **Animated Progress Bars** — Skill bars animate on scroll
- **Counter Animation** — Statistics count up on scroll
- **Testimonial Slider** — Auto-rotating testimonial carousel
- **Responsive Design** — Mobile-first, works on all devices
- **Loading Screen** — Animated preloader
- **Scroll Progress Bar** — Fixed top indicator
- **Sticky Navigation** — Glassmorphism navbar on scroll
- **3D Card Tilt** — Interactive hover effect on cards
- **SEO Optimized** — Meta tags, Open Graph, semantic HTML
- **Accessible** — ARIA labels, focus states, alt text
- **Zero Dependencies** — Pure vanilla HTML, CSS, JavaScript

## 🛠️ Customization

### Update Personal Info
Edit `index.html` to update name, title, contact info, and all section content.

### Update Profile Photo
Replace `assets/images/profile.jpg` with your own photo.

### Change Colors
Edit CSS custom properties in `style.css` under `:root`:
```css
:root {
  --accent-cyan: #00f0ff;
  --accent-purple: #a855f7;
  --accent-pink: #ff006e;
}
```

### Add Projects
Add new project cards in the `#projects` section following the existing card structure.
